# AI Video Maker | Automated Short-Form Content Generator 🎥

An automated video engine that transforms static images and audio tracks into engaging, social-media-ready vertical videos (9:16) using AI-powered transcription and cinematic effects.

## 🚀 Key Features
- **AI Word-Level Captions:** Uses OpenAI's **Whisper** model for high-accuracy transcription and word-synced overlays.
- **Dynamic Visuals:** Includes a "Dynamic Zoom" (Ken Burns) effect for a professional CapCut/VN aesthetic.
- **Cinematic Transitions:** Supports Crossfades, Dip to Black, and Hard Cuts.
- **Multi-lingual Support:** Optimized for English and Hindi (Devanagari) rendering.
- **Asynchronous Processing:** Built with Python threading to handle heavy video rendering in the background.

## 🛠️ Tech Stack
- **Backend:** Python, Flask
- **AI Model:** OpenAI Whisper (base)
- **Video Engine:** MoviePy
- **Database:** MySQL
- **Frontend:** HTML5, CSS3 (Grid/Flexbox), JavaScript (Vanilla)
- **Image Processing:** Pillow (PIL), ImageMagick
